# ImageVision AI

> Управление обработкой изображений с помощью естественного языка.  
> Natural Language → **LLM (Qwen / Llama / Mistral / ...)** → OpenCV

---

## 1. Описание проекта

**ImageVision AI** — десктопное приложение, которое позволяет управлять обработкой изображений, просто описывая желаемое действие на русском языке.

Вы пишете: _«Сделай чёрно-белым»_ — и программа понимает вас, применяет нужный фильтр OpenCV и показывает результат.

### Новые возможности (v2)

- 🔍 **Зум** — прокрутка колёсика мыши масштабирует изображение (от 10% до 1000%)
- 🖱️ **Панорамирование** — зажать ЛКМ + тянуть перемещает изображение при зуме
- 📐 **Открепляемые канвасы** — каждое из двух изображений можно «оторвать» в отдельное окно
- 🎯 **Зум к курсору** — точка под курсором остаётся на месте при масштабировании

## 4. Совместимость с LLM

Приложение использует стандартный **OpenAI-совместимый API** (`/v1/chat/completions`). Подойдёт **любая** модель, запущенная через LM Studio, Ollama (с адаптером) или другой сервер:

| Модель | Работает? |
|---|---|
| Qwen 2.5 / 3 (любой размер) | ✅ Отлично |

> Задача модели очень простая — вернуть один JSON из ~20 вариантов. Справятся даже модели от 3B параметров.

## 5. Почему это действительно искусственный интеллект

### 🧠 Большая языковая модель

В основе системы — **LLM** с миллиардами параметров, обученная на сотнях миллиардов токенов. Это полноценная нейросеть, а не набор правил.

### 💬 Понимание естественного языка

Пользователь **не обязан** знать правильный синтаксис команд. Модель одинаково успешно понимает:
- «Поверни на 90»
- «Хочу повернуть картинку вправо»
- «Сделай ее горизонтальной»
- «Rotate by 90 degrees»

### 🚫 Запрет keyword matching

В коде **нет ни одной** проверки вида `if "поверни" in text`. Весь смысл команды определяет модель, а OpenCV лишь исполняет структурированную инструкцию.

### 🔁 Схема Natural Language → LLM → Computer Vision

```
Текст на русском  →  [Нейросеть понимает смысл]  →  JSON  →  OpenCV
```

## 6. Список поддерживаемых команд

| № | Команда (пример) | JSON от LLM | OpenCV |
|---|---|---|---|
| 1 | «Поверни на 90 градусов» | `{"action":"rotate_90"}` | `cv2.rotate(...CLOCKWISE)` |
| 2 | «Поверни на 180» | `{"action":"rotate_180"}` | `cv2.rotate(...180)` |
| 3 | «Поверни на 270» | `{"action":"rotate_270"}` | `cv2.rotate(...CCW)` |
| 4 | «Отрази зеркально» | `{"action":"flip_h"}` | `cv2.flip(img, 1)` |
| 5 | «Переверни вертикально» | `{"action":"flip_v"}` | `cv2.flip(img, 0)` |
| 6 | «Сделай чёрно-белым» | `{"action":"grayscale"}` | `cv2.cvtColor(...GRAY)` |
| 7 | «Выдели красный канал» | `{"action":"red"}` | маскирование каналов |
| 8 | «Только зелёный» | `{"action":"green"}` | маскирование каналов |
| 9 | «Синий канал» | `{"action":"blue"}` | маскирование каналов |
| 10 | «Увеличь в два раза» | `{"action":"resize_up"}` | `cv2.resize(...×2)` |
| 11 | «Уменьши вдвое» | `{"action":"resize_down"}` | `cv2.resize(.../2)` |
| 12 | «Размой» | `{"action":"blur"}` | `cv2.GaussianBlur` |
| 13 | «Сделай ярче» | `{"action":"brightness"}` | `cv2.convertScaleAbs` |
| 14 | «Увеличь контраст» | `{"action":"contrast"}` | `cv2.convertScaleAbs` |
| 15 | «Найди границы» | `{"action":"edges"}` | `cv2.Canny` |
| 16 | «Инвертируй» | `{"action":"invert"}` | `cv2.bitwise_not` |
| 17 | «Резкость» | `{"action":"sharpen"}` | `cv2.filter2D` |
| 18 | «Сохрани» | `{"action":"save"}` | `cv2.imwrite` |

## 7. Управление канвасами

| Действие | Результат |
|---|---|
| Прокрутить колёсико мыши | Зум ±15% за тик, относительно курсора |
| Зажать ЛКМ + тянуть | Панорамирование изображения |
| Кнопка «⊞ Открепить» | Открывает канвас в отдельном окне |
| Кнопка «🔍 1:1» | Сбрасывает зум и позицию к центру |

## 8. Установка

### Предварительные требования

- Python 3.10+
- [LM Studio](https://lmstudio.ai/) с любой загруженной языковой моделью
- Запущенный локальный сервер LM Studio на `http://localhost:1234`

### Установка зависимостей

```bash
pip install -r requirements.txt
```

---

## 9. Запуск

```bash
python main.py
```

### Порядок работы

1. Запустите LM Studio → загрузите модель → включите сервер («Start Server»)
2. Запустите `python main.py`
3. В поле «СЕРВЕР LM STUDIO» введите адрес (по умолчанию `http://localhost:1234/v1`)
4. Нажмите «📂 Загрузить изображение»
5. Введите команду на русском языке и нажмите Enter или «▶ Выполнить»
6. Результат отображается в правом канвасе; используйте зум и открепление для удобного просмотра
