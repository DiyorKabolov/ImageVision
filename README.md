# ImageVision AI

> Управление обработкой изображений с помощью естественного языка.  
> Natural Language → **Qwen 3.5 9B** → OpenCV

---

## 1. Описание проекта

**ImageVision AI** — это десктопное приложение, которое позволяет управлять обработкой изображений, просто описывая желаемое действие на русском языке.

Вы пишете: _«Сделай чёрно-белым»_ — и программа понимает вас, применяет нужный фильтр OpenCV и показывает результат. Никаких кнопок на каждый фильтр, никакого изучения интерфейса.

## 4. Почему это действительно искусственный интеллект

### 🧠 Большая языковая модель

В основе системы лежит **Qwen 3.5 9B** — большая языковая модель с 9 миллиардами параметров, обученная на сотнях миллиардов текстовых токенов. Это полноценная нейросеть, а не набор правил.

### 💬 Понимание естественного языка

Пользователь **не обязан** знать правильный синтаксис команд. Модель одинаково успешно понимает:
- «Поверни на 90»
- «Хочу повернуть картинку вправо»
- «Сделай ее горизонтальной»
- «Rotate by 90 degrees»

Все эти фразы дают одинаковый результат, потому что **решение принимается нейросетью**, а не if-else цепочкой.

### 🚫 Запрет keyword matching

В коде **нет ни одной** проверки вида `if "поверни" in text`. Весь смысл команды определяет модель Qwen, а OpenCV лишь исполняет структурированную инструкцию.

### 🔁 Схема Natural Language → LLM → Computer Vision

```
Текст на русском  →  [Нейросеть понимает смысл]  →  JSON  →  OpenCV
```

Это классическая схема интеграции **NLP + CV**, используемая в production-системах.

## 5. Как работает Qwen

- Qwen запущена локально через **LM Studio** на порту `1234`
- Связь идёт по стандартному **OpenAI API**: `POST /v1/chat/completions`
- Системный промпт (в `prompts.py`) инструктирует модель возвращать **только JSON**
- Температура `0.1` — для стабильных, детерминированных ответов
- Приложение защищено от некорректного JSON: парсинг с несколькими fallback-стратегиями

---

## 6. Как работает OpenCV

После получения JSON-команды от Qwen, `ImageProcessor` вызывает соответствующий метод:

```python
{"action": "rotate_90"}   →  cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
{"action": "blur"}         →  cv2.GaussianBlur(img, (15, 15), 0)
{"action": "edges"}        →  cv2.Canny(gray, 50, 150)
{"action": "grayscale"}   →  cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
```

Каждое действие — отдельный метод класса `ImageProcessor`. Никакой логики выбора действия в OpenCV-коде нет — только исполнение.

## 9. Установка

### Предварительные требования

- Python 3.10+
- [LM Studio](https://lmstudio.ai/) с загруженной моделью **Qwen 3.5 9B**
- Запущенный локальный сервер LM Studio на `http://localhost:1234`

### Установка зависимостей

```bash
pip install -r requirements.txt
```

---

## 10. Запуск

```bash
python main.py
```

### Порядок работы

1. Запустите LM Studio и включите локальный сервер (кнопка «Start Server»)
2. Убедитесь, что модель Qwen загружена
3. Запустите `python main.py`
4. Нажмите «Загрузить изображение» и выберите файл
5. Введите команду на русском языке в поле ввода
6. Нажмите «Выполнить команду» или Enter
7. Наблюдайте результат справа, а JSON-ответ Qwen — в нижней левой панели
